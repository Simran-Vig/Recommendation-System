# Recommendation-System

**The Anime Suggestion Simulator** was created for the final project of CSC 111 at the University of Toronto.

It utilizes web-scraping and AI-based recommendation via community detection to recommmend the user relevant anime shows based on their input. 
Please see Project Report.pdf for further details.

Created by Lisa Ye, Iris Li, Anubha Joshi, and Simran Vig.
